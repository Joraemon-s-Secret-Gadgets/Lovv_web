removed from the active tree in commit `140a50a` ("feat(frontend): center align header
layout and clean up documents"). They are preserved here for historical traceability —
they document the original implementation intent behind the auth / chat / save / my-page /
planner flows and remain useful when reviewing why the current code behaves as it does.

Status: **archived reference, not actively maintained.** The shipped behavior is the
source of truth; where this archive disagrees with the code, the code wins.
